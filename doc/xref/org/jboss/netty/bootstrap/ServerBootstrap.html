46  /**
47   * A helper class which creates a new server-side {@link Channel} and accepts
48   * incoming connections.
49   *
50   * <h3>Only for connection oriented transports</h3>
51   *
52   * This bootstrap is for connection oriented transports only such as TCP/IP
53   * and local transport.  Use {@link ConnectionlessBootstrap} instead for
54   * connectionless transports.  Do not use this helper if you are using a
55   * connectionless transport such as UDP/IP which does not accept an incoming
56   * connection but receives messages by itself without creating a child channel.
57   *
58   * <h3>Parent channel and its children</h3>
59   *
60   * A parent channel is a channel which is supposed to accept incoming
61   * connections.  It is created by this bootstrap's {@link ChannelFactory} via
62   * {@link #bind()} and {@link #bind(SocketAddress)}.
63   * <p>
64   * Once successfully bound, the parent channel starts to accept incoming
65   * connections, and the accepted connections become the children of the
66   * parent channel.
67   *
68   * <h3>Configuring channels</h3>
69   *
70   * {@link #setOption(String, Object) Options} are used to configure both a
71   * parent channel and its child channels.  To configure the child channels,
72   * prepend {@code "child."} prefix to the actual option names of a child
73   * channel:
74   *
75   * <pre>
76   * {@link ServerBootstrap} b = ...;
77   *
78   * // Options for a parent channel
79   * b.setOption("localAddress", new {@link InetSocketAddress}(8080));
80   * b.setOption("reuseAddress", true);
81   *
82   * // Options for its children
83   * b.setOption("child.tcpNoDelay", true);
84   * b.setOption("child.receiveBufferSize", 1048576);
85   * </pre>
86   *
87   * For the detailed list of available options, please refer to
88   * {@link ChannelConfig} and its sub-types.
89   *
90   * <h3>Configuring a parent channel pipeline</h3>
91   *
92   * It is rare to customize the pipeline of a parent channel because what it is
93   * supposed to do is very typical.  However, you might want to add a handler
94   * to deal with some special needs such as degrading the process
95   * <a href="http://en.wikipedia.org/wiki/User_identifier_(Unix)">UID</a> from
96   * a <a href="http://en.wikipedia.org/wiki/Superuser">superuser</a> to a
97   * normal user and changing the current VM security manager for better
98   * security.  To support such a case,
99   * the {@link #setParentHandler(ChannelHandler) parentHandler} property is
100  * provided.
101  *
102  * <h3>Configuring a child channel pipeline</h3>
103  *
104  * Every channel has its own {@link ChannelPipeline} and you can configure it
105  * in two ways.
106  *
107  * The recommended approach is to specify a {@link ChannelPipelineFactory} by
108  * calling {@link #setPipelineFactory(ChannelPipelineFactory)}.
109  *
110  * <pre>
111  * {@link ServerBootstrap} b = ...;
112  * b.setPipelineFactory(new MyPipelineFactory());
113  *
114  * public class MyPipelineFactory implements {@link ChannelPipelineFactory} {
115  *   public {@link ChannelPipeline} getPipeline() throws Exception {
116  *     // Create and configure a new pipeline for a new channel.
117  *     {@link ChannelPipeline} p = {@link Channels}.pipeline();
118  *     p.addLast("encoder", new EncodingHandler());
119  *     p.addLast("decoder", new DecodingHandler());
120  *     p.addLast("logic",   new LogicHandler());
121  *     return p;
122  *   }
123  * }
124  * </pre>
125 
126  * <p>
127  * The alternative approach, which works only in a certain situation, is to use
128  * the default pipeline and let the bootstrap to shallow-copy the default
129  * pipeline for each new channel:
130  *
131  * <pre>
132  * {@link ServerBootstrap} b = ...;
133  * {@link ChannelPipeline} p = b.getPipeline();
134  *
135  * // Add handlers to the default pipeline.
136  * p.addLast("encoder", new EncodingHandler());
137  * p.addLast("decoder", new DecodingHandler());
138  * p.addLast("logic",   new LogicHandler());
139  * </pre>
140  *
141  * Please note 'shallow-copy' here means that the added {@link ChannelHandler}s
142  * are not cloned but only their references are added to the new pipeline.
143  * Therefore, you cannot use this approach if you are going to open more than
144  * one {@link Channel}s or run a server that accepts incoming connections to
145  * create its child channels.
146  *
147  * <h3>Applying different settings for different {@link Channel}s</h3>
148  *
149  * {@link ServerBootstrap} is just a helper class.  It neither allocates nor
150  * manages any resources.  What manages the resources is the
151  * {@link ChannelFactory} implementation you specified in the constructor of
152  * {@link ServerBootstrap}.  Therefore, it is OK to create as many
153  * {@link ServerBootstrap} instances as you want with the same
154  * {@link ChannelFactory} to apply different settings for different
155  * {@link Channel}s.
156  *
157  * @author <a href="http://www.jboss.org/netty/">The Netty Project</a>
158  * @author <a href="http://gleamynode.net/">Trustin Lee</a>
159  *
160  * @version $Rev$, $Date$
161  *
162  * @apiviz.landmark
163  */

168     /**

177     /**
178      * Creates a new instance with the specified initial {@link ChannelFactory}.
179      */

184     /**
185      * {@inheritDoc}
186      *
187      * @throws IllegalArgumentException
188      *         if the specified {@code factory} is not a
189      *         {@link ServerChannelFactory}
190      */
191     @Override
192     public void setFactory(ChannelFactory factory) {
193         if (factory == null) {
194             throw new NullPointerException("factory");
195         }
196         if (!(factory instanceof ServerChannelFactory)) {
197             throw new IllegalArgumentException(
198                     "factory must be a " +
199                     ServerChannelFactory.class.getSimpleName() + ": " +

205     /**
206      * Returns an optional {@link ChannelHandler} which intercepts an event

209      * @return the parent channel handler.
210      *         {@code null} if no parent channel handler is set.
211      */
212     public ChannelHandler getParentHandler() {
213         return parentHandler;
214     }

216     /**
217      * Sets an optional {@link ChannelHandler} which intercepts an event of
218      * a newly bound server-side channel which accepts incoming connections.

220      * @param parentHandler
221      *        the parent channel handler.
222      *        {@code null} to unset the current parent channel handler.
223      */
224     public void setParentHandler(ChannelHandler parentHandler) {
225         this.parentHandler = parentHandler;
226     }

228     /**
229      * Creates a new channel which is bound to the local address which was
230      * specified in the current {@code "localAddress"} option.  This method is
231      * similar to the following code:
232      *
233      * <pre>
234      * {@link ServerBootstrap} b = ...;
235      * b.connect(b.getOption("localAddress"));
236      * </pre>
237      *
238      * @return a new bound channel which accepts incoming connections
239      *
240      * @throws IllegalStateException
241      *         if {@code "localAddress"} option was not set
242      * @throws ClassCastException
243      *         if {@code "localAddress"} option's value is
244      *         neither a {@link SocketAddress} nor {@code null}
245      * @throws ChannelException
246      *         if failed to create a new channel and
247      *                      bind it to the local address
248      */
249     public Channel bind() {
250         SocketAddress localAddress = (SocketAddress) getOption("localAddress");
251         if (localAddress == null) {
252             throw new IllegalStateException("localAddress option is not set.");
253         }
254         return bind(localAddress);
255     }
256 
257     /**
258      * Creates a new channel which is bound to the specified local address.
259      *
260      * @return a new bound channel which accepts incoming connections
261      *
262      * @throws ChannelException
263      *         if failed to create a new channel and
264      *                      bind it to the local address
265      */
266     public Channel bind(final SocketAddress localAddress) {
267         if (localAddress == null) {
268             throw new NullPointerException("localAddress");
269         }
270 
271         final BlockingQueue<ChannelFuture> futureQueue =
272             new LinkedBlockingQueue<ChannelFuture>();
273 
274         ChannelPipeline bossPipeline;
275         ChannelHandler binder = new Binder(localAddress, futureQueue);
276         ChannelHandler parentHandler = getParentHandler();
277         if (parentHandler != null) {
278             bossPipeline = pipeline();
279             bossPipeline.addLast("binder", binder);
280             bossPipeline.addLast("userHandler", parentHandler);
281         } else {
282             bossPipeline = new StaticChannelPipeline(binder);
283         }
284 
285         Channel channel = getFactory().newChannel(bossPipeline);
286 
287         // Wait until the future is available.
288         ChannelFuture future = null;
289         boolean interrupted = false;
290         do {
291             try {
292                 future = futureQueue.poll(Integer.MAX_VALUE, TimeUnit.SECONDS);
293             } catch (InterruptedException e) {
294                 interrupted = true;
295             }
296         } while (future == null);
297 
298         if (interrupted) {
299             Thread.currentThread().interrupt();
300         }
301 
302         // Wait for the future.
303         future.awaitUninterruptibly();
304         if (!future.isSuccess()) {
305             future.getChannel().close().awaitUninterruptibly();
306             throw new ChannelException("Failed to bind to: " + localAddress, future.getCause());
307         }
308 
309         return channel;
310     }
311 
312     private final class Binder extends SimpleChannelUpstreamHandler {
313 
314         private final SocketAddress localAddress;
315         private final BlockingQueue<ChannelFuture> futureQueue;
316         private final Map<String, Object> childOptions =
317             new HashMap<String, Object>();
318 
319         Binder(SocketAddress localAddress, BlockingQueue<ChannelFuture> futureQueue) {
320             this.localAddress = localAddress;
321             this.futureQueue = futureQueue;
322         }
323 
324         @Override
325         public void channelOpen(
326                 ChannelHandlerContext ctx,
327                 ChannelStateEvent evt) {
328 
329             try {
330                 evt.getChannel().getConfig().setPipelineFactory(getPipelineFactory());
331 
332                 // Split options into two categories: parent and child.
333                 Map<String, Object> allOptions = getOptions();
334                 Map<String, Object> parentOptions = new HashMap<String, Object>();
335                 for (Entry<String, Object> e: allOptions.entrySet()) {
336                     if (e.getKey().startsWith("child.")) {
337                         childOptions.put(
338                                 e.getKey().substring(6),
339                                 e.getValue());
340                     } else if (!e.getKey().equals("pipelineFactory")) {
341                         parentOptions.put(e.getKey(), e.getValue());
342                     }
343                 }
344 
345                 // Apply parent options.
346                 evt.getChannel().getConfig().setOptions(parentOptions);
347             } finally {
348                 ctx.sendUpstream(evt);
349             }
350 
351             boolean finished = futureQueue.offer(evt.getChannel().bind(localAddress));
352             assert finished;
353         }
354 
355         @Override
356         public void childChannelOpen(
357                 ChannelHandlerContext ctx,
358                 ChildChannelStateEvent e) throws Exception {
359             // Apply child options.
360             e.getChildChannel().getConfig().setOptions(childOptions);
361             ctx.sendUpstream(e);
362         }
363 
364         @Override
365         public void exceptionCaught(
366                 ChannelHandlerContext ctx, ExceptionEvent e)
367                 throws Exception {
368             boolean finished = futureQueue.offer(failedFuture(e.getChannel(), e.getCause()));
369             assert finished;
370             ctx.sendUpstream(e);
371         }
372     }
